HANDLER-NAME
============
denkarium -- Logbuch + Gedanken-Sammler


BESCHREIBUNG
============
Der Denkarium-Handler verwaltet zwei Modi für Notizen und Reflexionen:

  denkarium  Freie Gedanken, Ideen und Reflexionen (flexibel kategorisierbar)
  logbuch    Chronologisches Tagebuch mit Sternzeit-Markierung

Alle Einträge werden in sqlite3 gespeichert (denkarium_entries Tabelle) und
können durchsucht, kategorisiert, bewertet und zu Tasks/Wiki befürdert werden.


OPERATIONEN
===========
bach denkarium write TEXT [--type=logbuch] [--cat=KATEGORIE] [--title=TITEL] [--mood=1-5]
    Neuen Eintrag schreiben. Default: type=denkarium, cat=notiz
    Beispiel: bach denkarium write "Idee für Automation" --type=denkarium --cat=idee

bach denkarium read [--type=logbuch|denkarium] [--cat=KATEGORIE] [--limit=N]
    Letzte N Einträge auflisten (default limit=10)
    Beispiel: bach denkarium read --type=logbuch --limit=5

bach denkarium search SUCHBEGRIFF
    In Inhalten, Titeln und Kategorien suchen (max. 20 Treffer)
    Beispiel: bach denkarium search "Automation"

bach denkarium brainstorm THEMA
    Neuen Brainstorm-Eintrag erstellen mit Template
    Beispiel: bach denkarium brainstorm "KI-Architektur"

bach denkarium promote ENTRY-ID task|wiki
    Denkarium-Eintrag zu Task oder Wiki befördern
    Beispiel: bach denkarium promote 42 task

bach denkarium categories
    Alle verwendeten Kategorien mit Eintragszählung auflisten

bach denkarium stats
    Gesamtstatistik: Anzahl, Zeitspanne, Beförderungen


BEISPIELE
=========
1. Schnelle Notiz (Standard-Denkarium):
   bach denkarium write "Morgen Backend-Refactoring planen"

2. Logbuch-Eintrag mit Bewertung:
   bach denkarium write "Produktive Session, viel erreicht" --type=logbuch --mood=5

3. Brainstorm starten:
   bach denkarium brainstorm "Neue Feature-Ideen"
   bach denkarium write "Caching-Layer hinzufügen" --cat=idee

4. Gedanken durchsuchen und zur Task befördern:
   bach denkarium search "wichtig"
   bach denkarium promote 15 task

5. Letzte Woche Logbuch-Einträge ansehen:
   bach denkarium read --type=logbuch --limit=20


DATEIEN
-------
  data/bach.db             SQLite-Datenbank (Tabelle: denkarium_entries)
  docs/help/denkarium.txt  Diese Hilfe-Datei


SIEHE AUCH
----------
  tasks    -- Aufgabenverwaltung (Ziel für promote-Operationen)
  wiki     -- Wiki-System (optionales Ziel für Beförderung)
  base     -- BaseHandler API (hub/base.py)


NOTIZEN
-------
- Sternzeit: Logbuch-Einträge ohne explizites --title erhalten "Sternzeit YYYY-MM-DD HH:MM"
- Kategorien: frei wählbar (notiz, idee, brainstorm, etc.)
- Mood-Rating: Optional, 1-5 Skala für Reflexion-Tracking
- Promote zu Task: Erstellt neuen Task mit standard priority=3, status=pending
- Promote zu Wiki: Markiert Eintrag, komplexe Wiki-Integration durch separaten Handler
