# llm-note

**llm-note** - локальный движок заметок для LLM-агентов. Он объединяет SQLite-журнал и текстовые блокноты без облачного сервиса и внешних runtime-зависимостей.

## Возможности

- Хранить заметки, записи журнала, категории, настроение и маркеры повышения.
- Использовать переносимые текстовые блокноты.
- Искать заметки через Python или CLI.
- Создавать записи для мозгового штурма и позже повышать их до задач, wiki-страниц или issues.
- Использовать пользовательские сообщения на шести языках.

## Быстрый старт

```bash
pip install -e .
llm-note --locale ru write "Проверить приватность перед релизом" --cat release
llm-note --locale ru search приватность
```
