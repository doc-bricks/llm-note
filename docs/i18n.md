# Internationalization

llm-note includes six standard message locales:

| Locale | File |
|---|---|
| German | `llm_note/locales/de.json` |
| English | `llm_note/locales/en.json` |
| Spanish | `llm_note/locales/es.json` |
| Simplified Chinese | `llm_note/locales/zh-Hans.json` |
| Japanese | `llm_note/locales/ja.json` |
| Russian | `llm_note/locales/ru.json` |

When adding a user-facing CLI message, add the same key to every locale file and run:

```bash
python -m pytest -q tests/test_i18n.py
```
